**1on1自律振り返りアプリ「Spiral Up Logger」開発設計書**

## **1\. プロダクトコンセプト**

* **ビジョン**: 土曜朝8時の1on1を「過去の報告」ではなく「未来の戦略会議」に変える。  
* **コアロジック**: 前週の「Will（実行する具体的な行動）」が翌週の「Goal（予定）」として自動継承されるスパイラル構造 。

* **開発環境**: anti GRAVITY (Google IDE)

## **2\. 振り返り観点とデータ構造**

アップロードされた記録に基づき、以下の3つのカテゴリを固定項目として定義します。

## **A. 学習（学問）**

* **目標（Goal）**: 学年10位〜20位以内の獲得 。

* **測定方法（Measurable）**:  
  * 学校ワーク、文法ワーク（国語）、数学トレーニング、完全学習（理科）、まんがワーク（社会）、ジョイフルワーク（英語）の進捗 。

  * 独自購入ワーク（ぴったりトレーニング、ハイクラス、教科書ワーク）の実施 。

* **評価**: 100点満点での自己採点と理解度の記録 。

## **B. サッカー**

* **目標（Goal）**: ソルティーロでのスタメン奪取、圧倒的なパフォーマンス、敵の壁（君臨する） 。

* **測定方法（Measurable）**:  
  * 自主練内容（リバウンダー、ランニング、ダッシュ、1on1）の回数・実施率 。

  * 練習・試合での手応え（インターセプト、裏対応、アジリティ等） 。

## **C. 生活**

* **目標（Goal）**: 安定的なメンタル、生活ルーティンの定着 。

* **測定方法（Measurable）**:  
  * 朝の起床ルーティン（座る→スクワット→飛ぶ） 。

  * 栄養摂取（マジデ/マジ・テin、牛乳、食事量） 。

  * ソーシャルディスタンス（家族との適切なコミュニケーション、暴言の回避） 。

**3\. アプリケーション機能要件**

## **① モーニング・エントリー機能（土曜 07:00〜）**

* **自動データ連携**: 前週の「W \- Will」に記載された「実行する具体的な行動」を、今週の「R \- Reality」のチェック項目として自動生成する 。

* **クイック入力**: 寝起きでも操作しやすいよう、数値入力（達成度）と、音声/トグルによる事実確認（やった/やらない）を優先。  
* **発明（Invention）の記録**: 課題解決のために自ら編み出した「工夫（例：計算ミスのための／入れ）」などをストックする 。

## **② 1on1サマリー生成機能（土曜 07:45〜）**

* **ROI分析（投資と回収）**: 「今週何に時間を投資し（Investment）、何が得られたか（Return）」を対照表として自動整形する 。

* **ボトルネック抽出**: 評価が低かった項目や「障害となっていること（Obstacles）」を強調表示し、1on1での相談議題を自動提案する 。

## **③ マンダラチャート連動**

* すべての振り返りが、最終的に「KENSEI\_マンダラチャート」の中心目標にどう近づいたかを視覚化する 。
